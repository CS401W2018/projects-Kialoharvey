  1. The form tag has an "action" attribute. What does it do? The HTML action Attribute is used to specify where the form data is to be sent to the server after the submission of the form.
  2. The form tag has a "method" attribute. What does it do? The method attribute in a form tag determines how form data is sent to the server upon submission, typically through either GET or POST.
  3. What is the difference between a "POST" and a "GET"? The main difference between GET and POST is in how they send data to the server and how secure they are. GET puts the data in the URL, so it’s visible in the address bar, which makes it less secure but convenient for things like searches or filters where privacy isn’t important. POST sends data in a hidden way, making it better for sensitive information like passwords.

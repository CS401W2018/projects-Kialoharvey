    Answer the following questions:
  • What is the difference between block tags and inline tags. Block tags create a block-level element that takes up the full width of the page or container.Whereas inline tags create inline elements that only take up as much space as they need.
  • What is the margin, border, and padding? Margin is the space outside the border of an element. It creates space between the element and other elements around it.Border is the line that goes around the edge of an element. You can control its thickness, color, and style. Padding is the space inside the element, between the content and the border. It adds space between the content and the border.
  • What are units of measurement you can use in CSS? Pixels(px), width, height, percent
  • When would you use a figure tag? The figure tag is used to group content, like images, charts, or code blocks, that is referenced in the main text of your webpage.

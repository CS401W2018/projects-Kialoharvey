1. Internet

The Internet is a global network of connected computers and devices that share information using common rules, mainly the Internet Protocol (IP)..

2. Web

The Web (short for World Wide Web, or WWW) is a system of interlinked documents, resources, and multimedia that are accessed over the Internet. It is one of the most popular services available on the Internet, but it's not the same as the Internet itself.

3. How are the internet and Web different?

the Internet is the physical infrastructure and global network, while the Web is a specific service that uses the Internet to deliver websites and web-based content.

4. ASCII and Unicode

The American Standard Code for Information Interchange (ASCII) is a system of digital codes that represent letters, numbers, and other symbols. It is commonly used to transfer text between computers

Unicode is a global system that gives each letter, number, and symbol a unique number. This system works with many languages and writing styles, making sure that text looks the same on different devices and programs

5. UFT - 8

UTF-8 is a system for encoding characters that can handle different lengths of text and is used for sending information electronically.

6. URL

A URL (Uniform Resource Locator) is the web address used to find a specific resource on the internet.

7. Domain Name

It shows which domain the address belongs to.
